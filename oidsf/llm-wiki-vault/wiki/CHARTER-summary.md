# Charter summary

OIDSF targets **public-interest OSINT** interchange: entity graphs, artifact-backed provenance, optional trust assessments, optional STIX/DISARM hooks.

**v1 default profile:** `graph-provenance` — entities, artifacts, statements, evidence links; optional source assessments.

**Versioning:** SemVer on `oidsf_version`; breaking changes bump major.
